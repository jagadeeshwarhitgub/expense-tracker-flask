# Expense Tracker Web App

A web-based expense tracker built using Python Flask and SQLite.

## Features
- User Registration & Login
- Add and Delete Expenses
- Total Expense Calculation
- Date Tracking
- Professional UI using Bootstrap

## Tech Stack
- Python
- Flask
- SQLite
- HTML, Bootstrap

## How to Run

1. Install Flask:
   pip install flask

2. Run app:
   python expense_app.py

3. Open browser:
   http://127.0.0.1:5000/
